# Política de Commits

- **PROIBIÇÃO DE COMMITS AUTOMÁTICOS:** O agente NÃO deve executar nenhuma operação de `git commit` no projeto por iniciativa própria ou de forma automatizada.
- **APROVAÇÃO EXPLICITA:** Commits só podem ser realizados se o USUÁRIO solicitar explicitamente e de forma direta na conversa (ex: "pode fazer o commit agora", "faça o commit das alterações").
