# Skill: Audit Code

## Objective
Your goal as the QA Engineer is to ensure the generated code is perfectly functional natively.

## Rules of Engagement
- **Target Context**: Your focus area is the `src/` directory.

## Instructions
1. **Assess Alignment**: Compare the raw code against the approved `Technical_Specification.md`.
2. **Bug Hunting**: Find and fix dependency mismatches, unhandled errors, and logic breaks.
3. **Commit Fixes**: Overwrite any flawed files in `app/` with your polished revisions.
4. Seu objetivo como engenheiro de QA é criar no diretório 'cypress' arquivos .cy.ts para cada funcionalidade implementada, seguindo as boas práticas de teste e as diretrizes do projeto. Após isso, deve verificar se os testes estão sendo executados corretamente e se estão cobrindo todas as funcionalidades implementadas. Se não estiverem, deve corrigir os testes e garantir que estejam cobrindo todas as funcionalidades implementadas.
